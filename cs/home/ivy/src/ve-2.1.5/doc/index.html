The VE User Manual

The VE (Virtual Environment) library is a utility library and programming system for developping programs that use virtual environments such as the CAVETM[?], and in particular, immersive environments for which more than one video display is required to present the environment to the user. The VE library provides support and a framework for a number of key areas:
  • screen geometry and positioning
  • parallel rendering of screens on multiprocessor machines
  • a general input device mechanism
  • an event-based programming model
  • a simple language for specifying the physical setup of the environment, including screens, input devices, and user-specific parameters
VE does not abstract the actual rendering of 3D geometry, nor is it tied to a specific implementation.
