BlueScript Reference

Packages

  • Core Procedures
        Core BlueScript functionality which is included in most BlueScript interpreters.
  • Hash Procedures
        Allows for the creation and manipulation of hash tables that map strings to arbitrary values or objects.
  • List Procedures
        Procedures for creating and manipulating lists.
  • Stream Objects
        Objects representing sequences of bytes (e.g. files) supporting a simple I/O model.
  • Standard Streams
        Common functions for creating streams (files, commands, stdio).
  • String Procedures
        Procedures for manipulating strings.
  • VE Environment
        Procedures for generating environments, audio setups, user profiles.
  • VE Input
        Procedures for creating input devices and manipulating input events.
  • VE Math Procedures
        An interface to vectors, quaternions, matrices, etc.
  • VE Miscellaneous
        Various VE support procedures.
